**How to install from source?**

On Windows:
````
git clone https://github.com/simargl/bplay
cd bplay
make win
````
On Linux:
````
git clone https://github.com/simargl/bplay
cd bplay
make install
````
Dependencies:
````
fltk 1.3.x
````
